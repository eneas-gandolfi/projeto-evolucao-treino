# Projeto Evolucao

Aplicativo PWA offline para consultar um plano semanal de treino.

## Como publicar no GitHub Pages

1. Publique este repositorio no GitHub.
2. Ative GitHub Pages usando a branch `main` e a pasta raiz.
3. Abra a URL gerada pelo GitHub Pages no celular.
4. Use a opcao do navegador para adicionar o app a tela inicial.

Depois da primeira carga completa, o app fica disponivel offline pelo service worker.
